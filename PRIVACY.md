# Privacy Policy

## Data Collection

This plugin does **not** collect, store, or transmit any user personal data on its own.

The plugin forwards the following information directly to your configured proxy endpoint:

- Prompt messages (user inputs and conversation history)
- Model parameters (max tokens, stop sequences)
- Tool definitions (if used)
- Your API key (sent as the `Authorization` header to authenticate with your proxy)

## Data Storage

The plugin stores no data. All credentials (API key, endpoint URL) are stored securely by Dify according to Dify's own privacy policy.

## Third-Party Services

The plugin communicates with the proxy endpoint URL you configure. Please review the privacy policy of your chosen proxy provider to understand how they handle your data.

## Contact

For privacy concerns, open an issue at https://github.com/zhuxiangyi/claude-proxy-plugin
